How It Works

DEADLINE is a live blockchain experiment on Solana. A countdown timer runs continuously, and every genuine on-chain transaction (buy/sell) extends the timer. If there are no transactions, the timer runs to zero, and the system ends.

Key Principles:

  • Only real blockchain data (no simulation, no fake volume)
  • No rewards, no buyback, no artificial mechanics
  • No manual intervention
  • Behavior is purely driven by market participants

Rules

The core idea is simple: The system only survives if users actively keep it alive. Every buy adds time, every sell removes time. When the timer hits zero, the experiment concludes. Always DYOR.
