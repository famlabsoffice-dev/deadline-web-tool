## 📘 FACEBOOK-SEITE SETUP-GUIDE (Integrity-konforme Implementierung)

### Phase 1: Persönliches Profil (Voraussetzung)
Falls noch kein Facebook-Konto existiert:
- Gehe auf facebook.com → "Neues Konto erstellen"
- **Echter Name** (Facebook sperrt Fake-Namen — Integrity Rule #1)
- **Echte E-Mail oder Handynummer**
- **Profilbild hochladen** (kein leeres Profil → senkt Trust-Score)
- **Handynummer verifizieren**
- ⚠️ **Warte 24–48h bevor du eine Seite erstellst** — neue Profile die sofort Pages erstellen werden oft eingeschränkt

### Phase 2: Facebook-Seite erstellen
1. Im Menü (links) → "Seiten" → "Neue Seite erstellen"
2. **Seitenname:** DEADLINE (kein $DIE24 im Namen — Dollar-Ticker im Seitennamen triggert oft automatische Überprüfung)
3. **Kategorie:** Kryptowährung oder Finanzen oder Community (wähle Community wenn du weniger Restriktionen willst)
4. **Bio:** (siehe Phase 3 unten)
5. **Profilbild + Coverbild hochladen**
6. "Seite erstellen" bestätigen

### Phase 3: Integrity-konforme Bio & Beschreibung
⚠️ **Facebook verbietet:** Renditeversprechen, "100x", "guaranteed profit", irreführende Token-Claims.

**Empfohlene Bio (unter 255 Zeichen):**
```
DEADLINE is a community-driven Web3 project. $DIE24 — built on transparency, decentralization, and real utility. No promises. Just progress.
```

**Über uns (längere Version für den "Info"-Tab):**
```
DEADLINE ($DIE24) is an independent Web3 community project. This page shares project updates, ecosystem news, and community content. Nothing posted here constitutes financial advice. Always do your own research (DYOR).
```
